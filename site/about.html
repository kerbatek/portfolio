About

I'm a developer based in Kraków, Poland, currently working as a Flutter developer on a commercial project while studying at WSZiB (Wyższa Szkoła Zarządzania i Bankowości w Krakowie). I hold a Technik Programista diploma.

I started with a single bare-metal Debian install running a few self-hosted services. As I wanted faster local network and NAS speeds, I upgraded to 2.5G networking and later to 10G - that's when I picked up a real interest in networking. I moved to Proxmox, started running services in Docker Compose, and eventually picked up a Supermicro platform to grow into.

From there the focus shifted to declarative configuration - first Docker Compose files stored in Git, and now transitioning into Kubernetes and infrastructure as code where possible, all provisioned with Terraform and deployed via ArgoCD.

That progression is how I discovered that I want to be a DevOps and cloud engineer. I love IaC, clustering, and designing infrastructure - both virtual and bare metal.
